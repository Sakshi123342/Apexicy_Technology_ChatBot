# Basic Chatbot - Python

This project is a simple **rule-based chatbot** developed using Python during my internship at **Apexicy Technology**.

##  Project Description

The chatbot interacts with users by responding to basic inputs such as greetings and simple questions. It uses predefined responses to simulate a basic conversation.

##  Features

* Responds to greetings like **hello**
* Answers simple questions like **how are you**
* Ends conversation when the user types **bye**
* Continuous interaction using a loop

##  Technologies Used

* Python
* Conditional Statements (if-elif)
* Loops
* Functions
* User Input/Output

## ▶ How to Run

1. Install Python on your system.
2. Download the project files.
3. Run the Python file:

```
python chatbot.py
```
